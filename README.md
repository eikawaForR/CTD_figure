(例図)2336航海でのSt.1で計測した塩分と深さのグラフ<br>

## Dependencies
+ numpy
+ pandas
+ xarray
+ matplotlib.pyplot
<br><br>

## Usage

### 1. データファイルの作成
各観測点のデータが入ったascファイルを入手，または作成する．
<br>

### 2. CTD_figure.pyを編集
<br>

```
#航海番号？と最大St名を入力
#Userが操作するのはここ↓
Vo_No=2336
max_St_No=3
```

<br>

### 3. run CTD_figure.py
<br>
